Audit the npm-facing package metadata for the local/tarball-first `v0.2.1` package surface.

This is not a public publish approval. Public npm publish remains deferred until a later decision.

## Files Policy

Included intentionally:

- `dist`: OpenCode plugin and CLI runtime.
- selected `scripts/*.mjs`: readiness, diagnostics, demo, and cleanup helpers.
- `.persona/harness.jsonc`: init template.
- `.persona/rules`: Java/Spring backend rule baseline.
- `docs/current/*.md`: release-facing support/readiness docs.
- `packages/shared-skills`: packaged reference material for the 1.0 direction.
- `README.md`: user-facing local/tarball flow.

Excluded intentionally:

- `experiments/`
- `.persona/evidence/`
- `.persona-test-fixtures/`
- source tests
- generated smoke projects
- raw phase run outputs

## Package Size

Latest dry-run observation:

- package size: approximately `1.3 MB`
- unpacked size: approximately `3.8 MB`
- total files: approximately `464`

The package is larger than a minimal Java-only plugin because `packages/shared-skills` is intentionally packaged as reference material. This is accepted for local/tarball-first `v0.2.1` and must be revisited before `1.0.0`.

## Dry-run And Install Verification

Verified commands:

```bash
npm publish --dry-run
npm pack --pack-destination /tmp/persona-npm-v021-check/pack
mkdir -p /tmp/persona-npm-v021-check/app
cd /tmp/persona-npm-v021-check/app
npm init -y
npm install -D /tmp/persona-npm-v021-check/pack/persona-harness-0.2.1.tgz
npx ph init
npx ph bearshell --help
npx ph bearshell node -e "console.log('v021-clean-install-ok')"
```

Observed:

- `npm publish --dry-run` completed without package metadata auto-correction warnings after `npm pkg fix`.
- Dry-run still reported that a real publish would require npm login; real publish remains deferred.
- Clean tarball install resolved `persona-harness@0.2.1`.
- `npx ph init` created `.persona/harness.jsonc`, `.persona/rules/`, and `.opencode/opencode.json`.
- `.persona/evidence/` was not copied by init.
- `npx ph bearshell --help` rendered the installed CLI help.
- `npx ph bearshell node -e "console.log('v021-clean-install-ok')"` executed successfully.
- Installed package included `packages/shared-skills`.
- Installed package included this metadata audit document.
