# Evidence Reviews

This directory contains A/B reviews, actual-run reviews, generated-run observations, regrades, and evidence summaries.

Evidence can inform decisions, but it is not active policy by itself. Promote only the resulting decision or rule into `../current/`.

Status: this directory is a repository documentation package only. It is not an
npm package named `evidence-review`, not a registry channel, and not part of the
Persona Harness version line. If a future standalone evidence-review tool is
needed, track it as a separate CLI/Release Prep task instead of inferring it from
this docs directory.
